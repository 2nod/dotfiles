# my persona karabiner config

My personal karabiner config built with [karabiner.ts](https://github.com/evan-liu/karabiner.ts)

## 編集場所

- ルール本体: `karabiner.ts`
- デバイス条件: `devices.ts`
- 補助関数: `utils.ts`

変更後は `deno task build` を実行して `karabiner.json` を更新します。

## Build

```bash
deno task build
```

or watch build

```bash
deno task watch
```
